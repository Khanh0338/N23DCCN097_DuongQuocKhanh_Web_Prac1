## Mô tả dự án
Bài thực hành 1: Xây dựng Product Listing Page sử dụng **Next.js** và **Tailwind CSS**.

## Tính năng đã hoàn thành
- ✅ Trang danh sách sản phẩm (`/`) - fetch dữ liệu từ FakeStore API
- ✅ Trang chi tiết sản phẩm (`/product/[id]`) - dynamic route
- ✅ Responsive layout (Mobile / Tablet / Desktop)
- ✅ Component tái sử dụng: Navbar, ProductCard, Button
- ✅ App Router (Next.js)

## Cách chạy dự án
```bash
npm install
npm run dev
```
Mở trình duyệt tại: http://localhost:3000

## Công nghệ sử dụng
- **Next.js** (App Router)
- **Tailwind CSS**
- **FakeStore API** - https://fakestoreapi.com/products
